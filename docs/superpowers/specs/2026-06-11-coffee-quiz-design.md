## מסכים

### 1. מסך פתיחה (Home)

- אייקון ☕, כותרת "כוסות קפה", תת-כותרת
- כפתור **"בואו נתחיל"** → מעבר למשחק
- כפתור **"רענן זיכרון"** → מעבר למסך רענון

### 2. מסך רענון זיכרון (Refresh)

רשימת סקאלת החוזק עם כוסות קפה ויזואליות:

| כוסות | מנה | הערה |
|---|---|---|
| ☕ חצי מלאה | מנה קצרה | הכי חלש |
| ☕ מלאה | מנה ארוכה | |
| ☕☕ שתי חצאי | כפול קצר | |
| ☕☕ שתי מלאות | כפול ארוך | הכי חזק |

- כפתור "בואו נתחיל ✓" → מעבר למשחק
- כפתור "→ חזרה" → חזרה לפתיחה

### 3. מסך המשחק (Quiz)

- כפתור "רענן זיכרון" קטן בפינה → מעבר למסך רענון
- בון + כפתורי תשובה

## לוגיקת מנות

### שמות מנות (שפת הקפה)

| Index | שם לתצוגה | הערה |
|---|---|---|
| 0 | חצי מנה | כפתור חמישי בלבד |
| 1 | מנה קצרה | |
| 2 | מנה ארוכה | |
| 3 | כפול קצר | (בעבר: "2 מנות קצרות") |
| 4 | כפול ארוך | (בעבר: "2 מנות ארוכות") |
| 5 | 3 מנות ארוכות | כפתור חמישי בלבד |

### ברירת מחדל לפי סוג קפה + כוס

| קפה | כוס | Index ברירת מחדל |
|---|---|---|
| הפוך / קפוצ'ינו / אמריקנו רגיל | פורצלן (default) | 1 — מנה קצרה |
| הפוך / קפוצ'ינו / אמריקנו רגיל | זכוכית | 2 — מנה ארוכה |
| הפוך / קפוצ'ינו / אמריקנו רגיל | TA | 2 — מנה ארוכה |
| הפוך / קפוצ'ינו / אמריקנו גדול | פורצלן (default) | 2 — מנה ארוכה |
| הפוך / קפוצ'ינו / אמריקנו גדול | זכוכית | 3 — כפול קצר |
| הפוך / קפוצ'ינו / אמריקנו גדול | TA | 4 — כפול ארוך |
| קפה קר / אמריקנו קר | זכוכית (default) | 3 — כפול קצר |
| קפה קר / אמריקנו קר | TA | 4 — כפול ארוך |

### מגביר חוזק

- **חלש** = index - 1
- **רגיל** (לא כתוב) = index ללא שינוי
- **חזק** = index + 1

### כללים מיוחדים

- **מפורק** → תמיד מנה ארוכה (index 2), ללא קשר לכוס/גודל/חוזק
- **מפורק** → לעולם לא יופיע עם חזק/חלש (האספרסו בנפרד מהחלב/מים)

### כפתור חמישי

מופיע **רק** כאשר התשובה הנכונה היא:
- **חצי מנה** (index 0) — הפוך/קפוצ'ינו/אמריקנו רגיל + פורצלן + חלש
- **3 מנות ארוכות** (index 5) — גדול + TA + חזק

## בנייה רנדומלית של הזמנה

### סוגי קפה

| שם | קטגוריה | גדלים | כוסות | כוס default |
|---|---|---|---|---|
| הפוך | hot | רגיל, גדול | פורצלן, זכוכית, TA | פורצלן |
| קפוצ'ינו | hot | רגיל, גדול | פורצלן, זכוכית, TA | פורצלן |
| אמריקנו | hot | רגיל, גדול | פורצלן, זכוכית, TA | פורצלן |
| קפה קר | cold | — | זכוכית, TA | זכוכית |
| אמריקנו קר | cold | — | זכוכית, TA | זכוכית |

- **אמריקנו:** מוסיף "חלב בצד" אוטומטית לבון
- **קפוצ'ינו = הפוך** מבחינת לוגיקת מנות

### פרמטרים אופציונליים

| פרמטר | ערכים | הסתברות |
|---|---|---|
| חוזק | חזק / חלש / רגיל | 25% / 25% / 50% |
| בסיס מים | כן / לא | 20% / 80% |
| נטול | כן / לא | 15% / 85% |
| מפורק | כן / לא | 15% / 85% |
| חלב | סויה / שקדים / שיבולת / דל / רגיל | 10% כל אחד / 60% |

## תצוגת הבון

### כלל תצוגת כוס

| מצב | מציג? |
|---|---|
| הפוך/קפוצ'ינו/אמריקנו בפורצלן | **לא** (default) |
| הפוך/קפוצ'ינו/אמריקנו בזכוכית | **כן** — "זכוכית" |
| הפוך/קפוצ'ינו/אמריקנו בTA | **כן** — "TA" |
| קפה קר/אמריקנו קר בזכוכית | **לא** (default) |
| קפה קר/אמריקנו קר בTA | **כן** — "TA" |

### סדר שורות הבון

```
[שם הקפה + גודל]   ← תמיד, גופן monospace גדול
[חזק / חלש]        ← רק אם לא רגיל
[בסיס מים]         ← רק אם כן
[נטול]             ← רק אם כן
[מפורק]            ← רק אם כן
[סוג חלב]          ← רק אם לא רגיל
[כוס]              ← לפי כלל תצוגת כוס
[חלב בצד]          ← רק לאמריקנו, תמיד
```

## עיצוב מסך (Quiz)

```
┌─────────────────────┐
│  רענן זיכרון        │  ← כפתור קטן
│                     │
│  הפוך גדול          │  ← bon-name, monospace
│  חזק                │  ← bon-line
│  שיבולת             │
│  זכוכית             │
│  - - - - - - - - -  │  ← קו מקווקו
│  [מנה קצרה][מנה ↵] │
│  [כפול קצר][כפול ↵]│
└─────────────────────┘
```

**spacer** דינמי בין הבון לכפתורים — הכפתורים עולים/יורדים לפי אורך הבון.

### מצבי כפתורים

| מצב | כפתור נכון | כפתור שגוי |
|---|---|---|
| לפני תשובה | לבן + מסגרת | לבן + מסגרת |
| תשובה נכונה | **ירוק** + "✓" | — |
| תשובה שגויה | **ירוק** + "✓" | **אדום** + "✗" |

### פלו אינטראקציה

1. בון רנדומלי מוצג
2. ברייסטה לוחצת על תשובה
3. פידבק ירוק/אדום מיידי
4. אחרי **1.5 שניות** → בון חדש אוטומטית
